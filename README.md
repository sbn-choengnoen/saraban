# ระบบงานสารบัญอิเล็กทรอนิกส์
## กองยุทธศาสตร์และงบประมาณ เทศบาลตำบลเชิงเนิน

---

## วิธีติดตั้ง (XAMPP)

### ขั้นตอนที่ 1 — วางไฟล์
1. เปิดโฟลเดอร์ `C:\xampp\htdocs\`
2. วางโฟลเดอร์ `saraban` ทั้งหมดลงไป
3. โครงสร้างจะเป็น: `C:\xampp\htdocs\saraban\`

### ขั้นตอนที่ 2 — สร้างฐานข้อมูล
1. เปิด **XAMPP Control Panel** → เปิด Apache และ MySQL
2. เปิดเบราว์เซอร์ไปที่ `http://localhost/phpmyadmin`
3. คลิก **"นำเข้า (Import)"** → เลือกไฟล์ `saraban.sql`
4. คลิก **"ดำเนินการ"**

### ขั้นตอนที่ 3 — เปิดใช้งาน
เปิดเบราว์เซอร์ไปที่:
```
http://localhost/saraban/
```

## โครงสร้างไฟล์

```
saraban/
├── index.php          ← หน้าทะเบียนหนังสือรับ
├── events.php         ← หน้าปฏิทินกิจกรรม
├── login.php          ← หน้าเข้าสู่ระบบ
├── logout.php         ← ออกจากระบบ
├── saraban.sql        ← ไฟล์สร้างฐานข้อมูล
└── includes/
    ├── config.php     ← ตั้งค่าฐานข้อมูล
    ├── navbar.php     ← แถบนำทาง
    └── footer.php     ← ส่วนท้าย
```

## ฟีเจอร์ของระบบ

### ทะเบียนหนังสือรับ
- เพิ่ม / แก้ไข / ลบหนังสือรับ
- ค้นหาตามเลขที่ / เรื่อง / หน่วยงาน
- กรองตามสถานะ (รับแล้ว / ดำเนินการ / เสร็จสิ้น)
- แสดงสถิติจำนวนหนังสือแยกตามสถานะ

### ปฏิทินกิจกรรม
- ปฏิทินรายเดือน เลื่อนดูเดือนก่อน/หลังได้
- เพิ่ม / ลบกิจกรรม แยก 3 ประเภท: ประชุม / กำหนดส่ง / กิจกรรม
- ตารางรายการกิจกรรมทั้งหมด

### ระบบ Login
- เข้าสู่ระบบด้วย username/password
- Session-based authentication
- ออกจากระบบได้

## หมายเหตุ
- รหัสผ่านเข้ารหัสด้วย `password_hash()` (bcrypt)
- ป้องกัน SQL Injection ด้วย Prepared Statements
- รองรับภาษาไทย (UTF-8)
